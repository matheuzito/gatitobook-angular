# Projeto Gatito Book - Angular

Projeto feito com o curso da Alura, vem a API que possui os procedimentos e registros do álbum de gatos e mais o projeto Front-end da aplicação Angular 11

Depois de clonar o projeto, abre no seu editor de texto (de preferência, Visual Studio Code) com o projeto aberto, insira o comando no terminal:
```
npm install
```
Ao fazer isso, ele vai instalar as dependências do projeto do Angular. Ao terminar de instalar, para ver o projeto funcionando só inserir o comando:

```
ng serve -o
```
Com o parâmetro ```-o``` significa ```--open``` ele vai abrir o navegador com o projeto rodando mas, para ver o funcionamento da API, Acesse a pasta chamado: ``api`` e depois abre o terminal digitando o comando:

``npm start``

Ele irá iniciar a API e assim, vai estar rodando os dois: A API e o Angular rodando do projeto.
